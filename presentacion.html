MotoMundo.pe

Proyecto web de tienda de motos

HTML semántico · CSS responsivo · Bootstrap · JavaScript

Proyecto web completo que integra HTML5 semántico, CSS con diseño responsivo usando Bootstrap 5, contenido multimedia y funcionalidad interactiva con JavaScript para manipulación del DOM.

Estructura del Proyecto

proyect-web-motos/
├── index.html        // Página principal
├── nosotros.html     // Quiénes somos
├── catalogo.html     // Catálogo con filtros
├── contacto.html     // Formulario + mapa
├── presentacion.html // Esta presentación
├── css/
│   └── styles.css      // Estilos personalizados
└── README.md
Tecnologías utilizadas
  • HTML5 semántico — <header>, <main>, <section>, <article>, <footer>, <nav>
  • Bootstrap 5 — Grid responsivo, componentes, utilidades
  • CSS3 personalizado — Animaciones, transiciones, diseño responsivo con media queries
  • JavaScript vanilla — Manipulación del DOM, eventos, filtros, validación
  • Multimedia — Imágenes, video embebido, mapa interactivo, iconos

Buenas prácticas
  • Etiquetas semánticas: <header>, <main>, <section>, <article>, <footer>, <nav>
  • Atributos ARIA: aria-label, aria-current="page", aria-labelledby para accesibilidad
  • Atributo loading="lazy" en imágenes para carga diferida
  • Texto alternativo (alt) descriptivo en todas las imágenes
  • Títulos jerárquicos: h1 → h2 → h3 correctamente anidados

JavaScript — Interacción DOM

catalogo.html — Filtro dinámico
// Array de objetos motos
const motos = [
  { nombre: "Yamaha R1", marca: "Yamaha",
    tipo: "Deportiva", precio: 18999, ... },
  ...
];

// Renderiza cards en el DOM
function mostrarMotos(lista) {
  const contenedor = getElementById("catalogo-motos");
  contenedor.innerHTML = lista.map(m => `...`).join("");
}

// Filtra según selects y actualiza DOM
function filtrarMotos() {
  const tipo = getElementById("filtro-tipo").value;
  const filtradas = motos.filter(m => {
    if (tipo !== "todos" && m.tipo !== tipo)
      return false;
    return true;
  });
  mostrarMotos(filtradas);
}

// Evento click en botón filtrar
addEventListener("click", filtrarMotos);
contacto.html — Validación de formulario
// Prevenimos envío por defecto
form.addEventListener("submit", function (e) {
  e.preventDefault();

  // Validación HTML5 nativa
  if (!form.checkValidity()) {
    form.classList.add("was-validated");
    return;
  }

  // Mostramos mensaje de éxito
  alertEl.classList.remove("d-none");
  alertEl.innerHTML = `¡Gracias ${nombre}!
    Mensaje recibido.`;

  form.reset();
  form.classList.remove("was-validated");
});
