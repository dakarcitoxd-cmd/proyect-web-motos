# MotoWorld - Tienda de Motos

Proyecto web de una tienda de motos desarrollado con **HTML5** y **Bootstrap 5**.

## Páginas

- **Inicio** — Landing page con hero, características y motos destacadas
- **Catálogo** — Listado de motos con filtros y cards dinámicas
- **Contacto** — Formulario de contacto con mapa embebido

## Deploy en Vercel

1. Conecta tu repositorio de GitHub en [vercel.com](https://vercel.com)
2. Vercel detectará automáticamente que es un proyecto HTML estático
3. Haz clic en **Deploy** y en segundos estará online

## Ver localmente

Abre cualquier archivo `.html` en tu navegador o usa un servidor local:

```bash
npx serve .
```
