# MotoMundo.pe - Tienda de Motos

Proyecto web completo de una tienda de motos desarrollado con **HTML5 semántico**, **CSS3 responsivo**, **Bootstrap 5** y **JavaScript vanilla**.

## Páginas

| Página | Descripción |
|--------|-------------|
| [`index.html`](index.html) | Landing page con hero, características, motos destacadas y video |
| [`nosotros.html`](nosotros.html) | Información de la empresa, historia, visión y valores |
| [`catalogo.html`](catalogo.html) | Catálogo con filtros dinámicos por tipo, marca y precio (JS) |
| [`contacto.html`](contacto.html) | Formulario de contacto con validación y mapa interactivo |
| [`presentacion.html`](presentacion.html) | Presentación del proyecto con explicación del flujo y código |

## Tecnologías

- **HTML5** — Etiquetas semánticas (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<nav>`)
- **CSS3** — Transiciones, animaciones, diseño responsivo con media queries
- **Bootstrap 5** — Grid responsivo, componentes, utilidades
- **JavaScript** — Manipulación del DOM, eventos, filtros, validación de formularios
- **Multimedia** — Imágenes optimizadas, video embebido de YouTube, mapa de Google Maps, iconos Bootstrap

## Ver localmente

```bash
npx serve .
```
